# yomuy_mobile

## YOMUY について About

「小説家になろう」に投稿されている小説をモバイル管理できるようにするアプリです。

## 主な機能 Functions

- [ ] 小説の検索
  - [x] 単語のみによる検索
  - [ ] 単語だけでなく詳細検索
- [ ] 小説の閲覧
  - [ ] オンラインでの閲覧
- [ ] 読書ログを記録
  - [ ] 検索して「本棚」へ登録
- [ ] 小説のダウンロード（オフライン読書）
